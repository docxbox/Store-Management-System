PURPOSE OF THE PROJECT:
-----------------------------------------------------------
This Java-based project simulates the basic operations of a store.
It demonstrates object-oriented programming (OOP) principles such as
classes, objects, and inheritance. The system includes support for:

- Managing departments within the store
- Managing retailers assigned to departments
- Running a GUI interface for simple user interaction

This project is educational and is ideal for beginners in Java.

-----------------------------------------------------------
HOW TO START THIS PROJECT:
-----------------------------------------------------------

1. Install Java JDK (version 17 or higher)
2. Open the project in BlueJ or any Java IDE (e.g., IntelliJ, Eclipse)
3. Open the file: Store_Gui.java
4. Compile and run the file to launch the graphical user interface
5. Interact with the store by adding departments and retailers
